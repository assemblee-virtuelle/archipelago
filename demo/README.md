Démo du calendrier intégré
À quoi sert cette application
Ceci est une démo d'une application qui a pour but de créer une iframe du calendrier du site archipelago / transiscope, pour que les utilisateurs puisses importer le calendrier sur leur site. L'application peu aussi etre utiliser pour obtenir un url prêt à être utiliser avec ou sans des filtres précis tel que thème ou organisation, ou avec les deux.
